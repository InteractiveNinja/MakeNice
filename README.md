# MakeNice
## Info
To make this thing work you need the library [clipboard](https://pypi.org/project/clipboard/)

## How to use

1. Copy Text into Clipboard (CTRL + C)
2. Dobble Click the Python File
3. Paste the Text in you Field you wanted it in
4. Done ~~be Happy~~
